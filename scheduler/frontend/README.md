# Scheduler Frontend Introduction

## Overview

The Scheduler Frontend consists of two primary components: the Client Interface and the Admin Dashboard.

## Client Inrerface

Designed for Connect Centre researchers, this module enables seamless booking of lab devices by selecting preferred time slots. Key functionalities include:​
* Intuitive device availability visualization​
* Flexible time slot selection with conflict detection​
* Reservation management and history tracking



## Admin Dashboard

Designed for Connect Centre administrators, this backend management portal provides rich control functionalities:
* Device inventory management (registration, configuration, status updates)
* Reservation oversight (approval, cancellation workflows, visualised booking status)
* Real-time testbed monitoring featuring critical metrics:
    * Power consumption 
    * Environment senstor data (temperaure/humidity)
